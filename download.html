Platform Compatibility

Windows: Native support for Windows 10 and Windows 11 (64-bit). The .exe installer provides seamless installation and updates.

Mac Users: Currently, Z Photo Selection is optimized for Windows. Mac users can run the software using compatibility solutions like:

  • Parallels Desktop or VMware Fusion (Virtual Machine)
  • Boot Camp (Dual boot with Windows)
  • Wine or CrossOver (Compatibility layer - may have limitations)
